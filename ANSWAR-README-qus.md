1️⃣ Difference between var, let, const:
ans: var can be re-declared and reassigned, let can be reassigned but not re-declared, const cannot be reassigned.

2️⃣ Spread operator (...):
ans: The spread operator is used to expand or copy elements of an array or object.

3️⃣ Difference between map(), filter(), forEach():
ans: map() returns a new array, filter() returns a new filtered array, forEach() only loops and returns nothing.

4️⃣ Arrow function:
ans: An arrow function is a shorter syntax for writing functions using =>.

5️⃣ Template literals:
ans: Template literals use backticks ` ` to insert variables in strings using ${}.
